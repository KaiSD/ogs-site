Learn to create games

This page is your starting point to learn ogstudio, a tool to create 3D games in the shortest time possible that run anywhere: web, desktop, and mobile.

Education process is built around GitHub Classroom. You can start any time by simply opening course URLs and following instructions in README. GitHub creates repositories for you inside OGStudio-Education organization.
